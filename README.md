# 🔴 Circle Fight

Một tựa game 2D thể loại giải đố - vượt chướng ngại vật được viết hoàn toàn bằng **C++** và thư viện **SFML**. Dự án áp dụng kiến trúc Lập trình Hướng đối tượng (OOP) và sử dụng `tmxlite` để đọc bản đồ thiết kế từ phần mềm Tiled.

## 🎮 Cách chơi
- Sử dụng **Phím mũi tên** hoặc **W, A, S, D** để di chuyển nhân vật (Khối vuông đỏ).
- **Mục tiêu:** Né tránh các chướng ngại vật (Khối tròn xanh) và đi đến Vùng đích (Màu hồng) an toàn.
- Game có hệ thống bấm giờ và lưu Kỷ lục (Highscore) cho người chơi hoàn thành 3 màn chơi nhanh nhất!

## 🛠️ Công nghệ sử dụng
- **Ngôn ngữ:** C++ (OOP)
- **Đồ họa & Cửa sổ:** SFML (Simple and Fast Multimedia Library)
- **Thiết kế màn chơi:** Tiled Map Editor (`.tmx`) & thư viện `tmxlite`

## 🚀 Hướng dẫn cài đặt và chạy (Build)
1. Cài đặt trình biên dịch MinGW (hỗ trợ C++17 trở lên).
2. Tải và cấu hình thư viện SFML.
3. Mở thư mục dự án bằng VS Code.
4. Nhấn `Ctrl + Shift + B` để build mã nguồn.
5. Chạy file `circle_fight.exe` để bắt đầu trò chơi.
